## Notes

- **Headline**: italicize words with `*italic*` in DA.live — they render in neon green
- **Image**: wide landscape photo recommended (2000 × 700px+); auto-darkened via CSS `filter: brightness(0.65)`
- Primary CTA gets the neon button style; secondary CTA gets a white outline button
